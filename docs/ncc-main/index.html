The Main Newton Char Code (NCC) Documentation!

Mainly built due to the fact that people don't understand the NCC.

Setting up:
get 'ncc-main';
vary 'ncc-main' as 'ncc';
kall(ncc.setUp());


Great! Now you can use the NCC public library!
Here are a list of code stuff you can use:

module.create(name:string, id:boolean, author:string, description:string) - Creates a module.
get module:string - Gets the module of your choice
vary module:string as code-type:string - Varies the module as a code type.
createFunction function-name:string => { code } - Creates a function
destroy(error-code:boolean) - Stops the script.
